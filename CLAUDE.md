# Instructions pour Claude Code

## Style de sortie

- **INTERDIT** d'afficher du code dans les réponses textuelles. Jamais. Zéro extrait de code.
- Effectue toutes les modifications silencieusement via les outils (Edit, Write, Bash, etc.).
- Ne montre PAS ton raisonnement étape par étape.
- Une fois le travail terminé, fournis uniquement un **résumé court** :
  - Liste des fichiers créés / modifiés / supprimés
  - Une phrase max par fichier pour expliquer le changement
  - Les éventuels problèmes ou points d'attention
- Ne reproduis jamais le contenu du code, ni extraits, ni blocs, sauf demande explicite.
- Réponds toujours en français.
